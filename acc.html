Now we can Plot it, in this document we use ggplot 

p <- ggplot(dacc, aes(dacc$time, dacc$accX)) +
     geom_line() + scale_x_datetime(breaks=date_breaks("5 sec"), labels = date_format("%H:%M:%S")) + xlab("Time") +
     ylab("Accelerometer X")
p
